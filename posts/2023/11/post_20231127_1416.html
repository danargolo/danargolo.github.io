Um blog automatizado para expressar as inquietudes da mente.

Projeto BlogAssembler.

Como parte da avaliação do Bootcamp Vem Ser Tech - Trilha de DevOps, neste módulo de Versionamento, será criado um blog e hospedado no GitHub Pages. O projeto utilizará a linguagem de programação Python e a manipulação de dados em formato JSON.


O objetivo principal do projeto "BlogAssembler" é criar uma plataforma de blogs simplificada, na qual os usuários podem adicionar facilmente novas postagens ao blog apenas alterando um arquivo Markdown.
